## How to install/run this

### Linux

1. Install Firefox

```console
sudo apt update
sudo apt install firefox
```

2. Install Python
```console
sudo apt install python3
```

3. Install Deno

```console
curl -fsSL https://deno.land/install.sh | sh
```

4. Install FFmpeg
```console
sudo apt install ffmpeg
```

### Windows

1. Install [Firefox](https://www.firefox.com/en-US/browsers/desktop/windows/)

2. Install [Python](https://www.python.org/downloads/)

3. Install Deno (open powershell)
```powershell
irm https://deno.land/install.ps1 | iex
```

5. Install [FFmpeg](https://windowsloop.com/install-ffmpeg-windows-10/)

### MacOS 

1. Install [Firefox](https://www.firefox.com/en-US/browsers/desktop/mac/)

2. Install [Python](https://www.python.org/downloads/)

3. Install FFmpeg (Brew required)

Install Brew (open the terminal)

```console
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
Install ffmpeg

```console
brew install ffmpeg
```

4. Install Doner
```console
curl -fsSL https://deno.land/install.sh | sh
```
### MacOS (Brew)

1. Install Brew

```console
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
2. Install Firefox

```console
brew update
brew install --cask firefox
```

3. Install Python
```console
brew install python
```

4. Install FFmpeg
```console
brew install ffmpeg
```

## Running and installing The discord bot

5. Install Python packages
```python
pip install -r requirements.txt
```
6. change example.env to .env and add your configuration >:3

7. Now run the bot.py file
```python
python bot.py
```
