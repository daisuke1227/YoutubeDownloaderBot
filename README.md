## How to install/run this

### Linux

1. Install Firefox

```console
sudo apt update
sudo apt install firefox
```

2. Install Python
```console
sudo apt install python3
```

3. Install Deno

```console
curl -fsSL https://deno.land/install.sh | sh
```

4. Install FFmpeg
```console
sudo apt install ffmpeg
```

5. Install aria2c
```console
sudo apt install aria2
```

### Windows

1. Install [Firefox](https://www.firefox.com/en-US/browsers/desktop/windows/)

2. Install [Python](https://www.python.org/downloads/)

3. Install Deno (open powershell)
```powershell
irm https://deno.land/install.ps1 | iex
```

5. Install [FFmpeg](https://windowsloop.com/install-ffmpeg-windows-10/)

6. Install aria2c
```powershell
winget install aria2.aria2
```

### MacOS 

1. Install [Firefox](https://www.firefox.com/en-US/browsers/desktop/mac/)

2. Install [Python](https://www.python.org/downloads/)

3. Install FFmpeg (Brew required)

Install Brew (open the terminal)

```console
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
Install ffmpeg

```console
brew install ffmpeg
```

4. Install Doner
```console
curl -fsSL https://deno.land/install.sh | sh

5. Install aria2c

```

### MacOS (Brew)

1. Install Brew

```console
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
2. Install Firefox

```console
brew update
brew install --cask firefox
```

3. Install Python
```console
brew install python
```

4. Install FFmpeg
```console
brew install ffmpeg
```

5. Install aria2c (optional, for faster downloads)
```console
brew install aria2
```
